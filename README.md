# A&W Pressure Washing Calculator

**Professional mix and surface calculations for pressure washing and soft wash crews — free, fast, and works offline.**

---

## What This App Does

The A&W Pressure Washing Calculator is a field tool built specifically for professional soft wash and pressure washing technicians. It eliminates mental math on the job and gives your crew accurate, repeatable calculations every time.

Two core calculators are included:

| Calculator | What It Does |
|---|---|
| **Mix Calculator** | Calculates exact SH, water, and surfactant volumes for any batch size using the C₁V₁ = C₂V₂ dilution formula |
| **Surface Calculator** | Calculates washable square footage for houses, roofs, driveways, and fences — then estimates how much mix you'll need |

A full **Chemical Safety** reference is also built in, covering PPE requirements, SDS data for Sodium Hypochlorite, incompatible chemicals, and first aid procedures.
